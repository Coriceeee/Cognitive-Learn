# Cognitive Learn AI — Starter MVP

## Chạy dự án
```bash
npm install
cp .env.example .env
npm run dev
```

## Phạm vi bước 1
- Dashboard học sinh
- Kiểu dữ liệu hồ sơ học tập và chỉ số nhận thức
- Hàm tính SCI, MAS, CSL phiên bản baseline
- Cấu hình Firebase qua biến môi trường

## Bước tiếp theo
